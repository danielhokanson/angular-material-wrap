{{ selectedDirective.name }} Directive API

Directive API Overview

AmwTooltipDirective

Displays custom tooltips with configurable positioning and styling.

  • Selector: [amwTooltip]
  • Inputs: amwTooltip: string, position: 'top' | 'bottom' | 'left' | 'right'
  • Usage: Add to any element to show a tooltip on hover

AmwClickOutsideDirective

Detects and handles clicks outside of an element.

  • Selector: [amwClickOutside]
  • Outputs: (amwClickOutside): EventEmitter<MouseEvent>
  • Usage: Useful for closing dropdowns, modals, or menus

AmwAutoFocusDirective

Automatically focuses an element when the component is initialized.

  • Selector: [amwAutoFocus]
  • Inputs: amwAutoFocus: boolean (default: true)
  • Usage: Apply to form inputs for immediate focus

AmwCopyToClipboardDirective

Copies text to the clipboard when an element is clicked.

  • Selector: [amwCopyToClipboard]
  • Inputs: amwCopyToClipboard: string
  • Outputs: (copied): EventEmitter<boolean>
  • Usage: Attach to buttons to enable copy functionality
