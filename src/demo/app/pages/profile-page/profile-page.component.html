Key Features

  • Tabbed navigation for organizing profile sections
  • Reactive forms for personal information with validation
  • Profile picture upload and management
  • Password change functionality with confirmation
  • User preferences with theme and language selection
  • Notification toggles using switches
  • Action buttons for saving and resetting

Best Practices

  • Use reactive forms for complex forms with validation
  • Use ngModel for simple preference toggles
  • Group related settings with clear headings
  • Provide visual feedback when saving changes
  • Validate passwords match before submission
  • Consider auto-save for preferences

Components Used

  • AmwTabsComponent / AmwTabComponent - For organizing profile sections
  • AmwCardComponent - For grouping related content
  • AmwDividerComponent - For separating setting groups
  • AmwInputComponent - For text inputs and password fields
  • AmwSelectComponent - For dropdown selections
  • AmwSwitchComponent - For toggle preferences
  • AmwButtonComponent - For save/reset actions
  • AmwIconComponent - For profile avatar placeholder

Page Structure

  • Header: Page title and description
  • Tab Navigation: Personal Info, Account, Preferences tabs
  • Personal Info: User details form with validation
  • Account: Profile picture and password management
  • Preferences: Theme, language, and notification settings

Customization

  • Add additional profile fields as needed
  • Implement profile picture upload with file input
  • Add two-factor authentication settings
  • Include account deletion or deactivation options
  • Add linked accounts (social login) management
  • Implement session management and active devices
