Detail/View Page Demo

This demo showcases the AMW Detail Page component with comprehensive employee information, including personal details, professional information, and related data.

Basic detail view with essential information and actions.

Advanced detail view with all features: print, share, and additional actions.

Demo Information

Features Demonstrated:

  • Tabbed and single-section content layouts
  • Field-level actions (copy, link)
  • Custom field renderers and components
  • Related data sections (tables, lists)
  • Action buttons (edit, delete, print, share)
  • Real-time updates support
  • Responsive design

Data Structure:

The demo uses a comprehensive employee record with:

  • Personal information (name, email, phone, address)
  • Professional details (department, role, manager, team)
  • Compensation and benefits
  • Skills and performance metrics
  • Related projects and performance reviews

Custom Features:

  • Clickable email and phone links
  • Copyable field values
  • Custom field renderers for complex data
  • Color-coded status indicators
  • Collapsible related data sections
