Pipes Demo

Interactive demonstration of data transformation pipes.

Currency Pipe

Format numbers as currency with different locales and display options.

Basic: {{ 1234.56 | amwCurrency }}
EUR: {{ 1234.56 | amwCurrency:'EUR' }}
JPY: {{ 1234.56 | amwCurrency:'JPY' }}
Custom digits: {{ 1234.56 | amwCurrency:'USD':'symbol':'1.0-0' }}

Date Pipe

Format dates with various display options and locales.

Short: {{ currentDate | amwDate:'short' }}
Medium: {{ currentDate | amwDate:'medium' }}
Long: {{ currentDate | amwDate:'long' }}
Date only: {{ currentDate | amwDate:'date' }}
Time only: {{ currentDate | amwDate:'time' }}

Text Transform Pipe

Transform text with various case and format options.

Uppercase: {{ textValue | amwTextTransform:'uppercase' }}
Lowercase: {{ textValue | amwTextTransform:'lowercase' }}
Capitalize: {{ textValue | amwTextTransform:'capitalize' }}
Title Case: {{ textValue | amwTextTransform:'titlecase' }}
Camel Case: {{ textValue | amwTextTransform:'camelcase' }}
Kebab Case: {{ textValue | amwTextTransform:'kebabcase' }}
Snake Case: {{ textValue | amwTextTransform:'snakecase' }}
