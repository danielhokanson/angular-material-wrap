Digit Info Format:

Transform Details

Case Transformations:

  • uppercase: Converts entire string to UPPERCASE
  • lowercase: Converts entire string to lowercase
  • capitalize: First character uppercase, rest lowercase
  • titlecase: First character of each word uppercase

Programming Styles:

  • camelcase: Removes spaces, first word lowercase, subsequent words capitalized (helloWorld)
  • kebabcase: Replaces spaces/underscores with hyphens, all lowercase (hello-world)
  • snakecase: Replaces spaces/hyphens with underscores, all lowercase (hello_world)

Conversion Intelligence:

  • Detects existing formatting (camelCase, PascalCase, etc.)
  • Handles transitions between uppercase and lowercase letters
  • Preserves word boundaries when converting between styles

{{ selectedPipe.name }} Pipe API

Pipe API Overview

AmwCurrencyPipe

Formats numbers as currency values with internationalization support.

  • Signature: transform(value: number, currencyCode?: string, display?: 'symbol' | 'code' | 'name', digitsInfo?: string, locale?: string): string
  • Parameters: value, currencyCode (default: 'USD'), display (default: 'symbol'), digitsInfo, locale

AmwDatePipe

Formats dates with customizable patterns and timezone support.

  • Signature: transform(value: Date | string | number, format?: string, timezone?: string, locale?: string): string
  • Parameters: value, format (default: 'mediumDate'), timezone, locale

AmwTextTransformPipe

Transforms text case with multiple formatting options.

  • Signature: transform(value: string, mode: 'uppercase' | 'lowercase' | 'capitalize' | 'titlecase'): string
  • Parameters: value, mode (required)
