Key Features

  • Full Calendar with month, week, and day views
  • Mini Calendar for compact layouts
  • Drag-and-drop event management
  • Strongly typed events with TypeScript generics
  • Customizable event colors via configuration
  • Comprehensive event handlers for all interactions
  • Responsive design for all screen sizes

Best Practices

  • Define a typed interface for your event data
  • Use eventColor function for dynamic color coding
  • Handle all event change types (create, update, delete, move)
  • Provide visual feedback using notifications
  • Use Mini Calendar for dashboard widgets
  • Configure time slots based on your use case

Components Used

  • AmwCalendarFullComponent - Full-screen calendar with multiple views
  • AmwCalendarMiniComponent - Compact calendar for sidebars and dashboards
  • AmwTabsComponent / AmwTabComponent - For organizing demo sections
  • AmwCardComponent - For grouping content
  • AmwDividerComponent - For separating sections
  • AmwButtonComponent - For demo controls
  • AmwIconComponent - For feature icons

Calendar Views

  • Month View: Overview of the entire month with event indicators
  • Week View: Detailed view of a week with time slots
  • Day View: Detailed view of a single day with time slots
  • Mini Calendar: Compact single day or week view

Event Operations

  • Create: Click or double-click on empty cell to create event
  • Edit: Click on event to open edit dialog
  • Delete: Use delete button in event popup
  • Move: Drag and drop events to new dates/times
  • Resize: Drag event edges to change duration

Customization

  • Configure start/end hours for time views
  • Set time slot intervals (15, 30, 60 minutes)
  • Customize event colors based on data
  • Show/hide weekends
  • Enable/disable specific features (edit, delete, drag)
  • Set default event duration
