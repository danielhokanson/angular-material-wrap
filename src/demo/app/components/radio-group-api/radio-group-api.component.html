Usage Examples

Common usage patterns and examples for the amw-radio-group component.

Basic Usage 
<amw-radio-group
  [options]="options"
  label="Choose an option"
  [(value)]="selectedValue"
  (valueChange)="onValueChange($event)">
</amw-radio-group>
With Configuration Object 
<amw-radio-group
  [config]="radioGroupConfig"
  [(value)]="selectedValue">
</amw-radio-group>
Reactive Forms 
<form [formGroup]="myForm">
  <amw-radio-group
    [options]="options"
    formControlName="myField"
    [required]="true">
  </amw-radio-group>
</form>
