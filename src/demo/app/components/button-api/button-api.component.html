Usage Notes

  • Content Projection: Use either text input or ng-content for button text
  • Icons: Use Material Design icon names for the icon property
  • Accessibility: Always provide ariaLabel for icon-only buttons
  • Loading State: When loading is true, the button shows a spinner and is disabled
  • FAB Buttons: FAB variants are circular and should be used for primary actions
