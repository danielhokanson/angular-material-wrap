Usage Notes

  • Two-way Binding: Use [(checked)] for easy two-way data binding
  • Form Integration: The component implements ControlValueAccessor for seamless form integration
  • Validation: Use the hasError and errorMessage properties for validation feedback
  • Accessibility: The component includes proper ARIA labels and keyboard navigation
  • Label Positioning: Labels can be positioned before or after the switch
  • Size Variations: Three sizes available for different use cases
  • Color Themes: Supports Material Design color themes (primary, accent, warn)
