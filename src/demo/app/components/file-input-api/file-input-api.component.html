Usage Notes

  • File Selection: Use [(selectedFiles)] for two-way data binding
  • Form Integration: The component implements ControlValueAccessor for seamless form integration
  • Validation: Use the hasError and errorMessage properties for validation feedback
  • Drag & Drop: Built-in drag and drop functionality with visual feedback
  • File Types: Support for MIME types and file extensions
  • Size Validation: Automatic file size validation with customizable limits
  • File Management: Built-in file list with remove functionality
  • Accessibility: Proper ARIA labels and keyboard navigation
