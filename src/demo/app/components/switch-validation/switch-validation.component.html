Validation Rules

  • Required Switches: Notifications, Auto Save, and Terms must be enabled
  • Optional Switches: Dark Mode can be enabled or disabled
  • RequiredTrue Validator: Ensures switches are in the "on" position
  • Form State: Form is invalid if any required switches are off
  • Error Display: Shows validation errors below each switch
